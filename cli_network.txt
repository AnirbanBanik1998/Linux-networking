                                               15 Commands To Get Started With Networking 

1 'ip a' or 'ip addr' :Check the ip adresses of  all the interfaces detected in the network.

2 'ip l' or 'ip link' :Display information for all interfaces.

3 'ip r' or 'ip route'Check the ip address of the default gateway(if at all specified) ,mainly for troubleshooting purposes.

4 'ip addr add <ip> dev <interface name>' :Assign ip to a specific interface.

5 'ip addr del <ip> dev <interface name>' :Remove an ip address of a specific interface.

6 'ip link set <interface name> up' :Enable a network interafce.

7 'ip link set <interface name> down' :Disable a network interface.

8 'nmcli dev' :List devices,their type,state and connection status.

9 'nmcli dev wifi connect <ssid> password <pswd>' :Using nmcli as network manager to connect wifi.

10 'nmcli con <interface name> up' :Set up coonnection for a specified interface.

11 'nmcli con <interface name> down' :Disconnect a specified interface.

12 'nmcli con reload' :Reload connection after modifications.

13 'nmtui' :Text user interface as network manager.It gives optons for adding, editing(Add a hostname,ip adress,dns server,gateway etc) or deleting a network.

14 'vi /etc/sysconfig/network-scripts/ifcfg-<connection/interface name>' :Edit configuration file of a specified network.

15 'ping <ip>' :Check connectivity of specified network.
